Personal Chatbot Assistant Using Python

This chatbot is designed to act as a personal assistant, schedule reminder and also keep company in times of boredom.

Photo by Alex Knight on Unsplash

Introduction

In this project I created a chatbot that could also act as a personal assistant. Also, the chatbot should be able to update its database with new chats or questions it is asked. This will help in improving the model as the App learns. the project features machine learning tools such as Scikit learn, NLTK, Pickle and some Python Libraries.

Steps covered in this Project

The following machine learning steps will be carried out to in creating this chatbot:

  • Download and Import necssary Python modules.
  • Preprocessing data.
  • Create training and test data./li>
  • Building the ML model.
  • Predicting Response
  • Save new data to the dataset.
